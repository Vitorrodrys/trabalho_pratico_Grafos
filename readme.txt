
README
Este repositório contém trabalhos desenvolvidos na disciplina de Estrutura de Dados 2, com foco em grafos. Os projetos incluem a implementação de sistemas relacionados a rotas e voos, destacando o cálculo da menor rota em termos de arestas usando BFS, o cálculo de Dijkstra para determinar o menor voo de um aeroporto para outro, o uso de DFS para descobrir um caminho que passa por todos os vértices, o algoritmo de Prim para gerar a árvore geradora mínima e um algoritmo para gerar uma coloração no grafo.

Algoritmos Implementados
BFS (Busca em Largura)
O algoritmo BFS é implementado no arquivo /graphs/graph_routes com o nome de grt_bfs.

DFS (Busca em Profundidade)
O algoritmo DFS pode ser encontrado em /graph/graph_routes com o nome de grt_dfs.

Dijkstra
O cálculo de Dijkstra é realizado no arquivo /graph/flights_digraph com o nome de generate_dijkstra.

Prim
O algoritmo de Prim está implementado em /graphs/graph_routes com o nome de grt_prim.

Coloração
A geração de coloração no grafo é realizada em /graphs/graph_routes com o nome de grt_coloring.

Estruturas de Dados
Foram utilizadas estruturas de dados genéricas para diversos procedimentos, como a associação de apelidos de aeroportos aos seus correspondentes números de vértice no grafo. Além disso, informações completas sobre voos foram vinculadas a seus apelidos correspondentes. Essas estruturas estão mais detalhadamente explicadas no relatório.

Estrutura do Projeto
/graph/searchs: Contém os TADs (Tipos Abstratos de Dados) que auxiliam na passagem de informações sobre a resposta e realizam a busca nos algoritmos mencionados.
